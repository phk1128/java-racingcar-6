## 🎯 기능 구현 목록

### 자동차 목록 저장 (Cars)
- [X] 입력 받은 자동차 목록을 Map타입으로 저장
- [X] 예외처리
    - 자동차가 2대 미만 인경우
    - 자동차의 이름이 5글자를 초과한 경우
    - 자동차 이름이 중복인 경우

### 경주 횟수 저장 (CarsGameCount)
- [X] 입력 받은 횟수를 int타입으로 저장
- [X] 예외처리
  - 숫자가 아닌 경우


### 경주 결과 저장 (CarsGameResult)
- [X] 랜덤 숫자 생성시 4미만 값이 나올 경우 value값에 -1 (후진)
- [X] 경주의 결과를 Map타입으로 저장
